Enhancement One

Software Engineering and Design

Complete UI redesign and improved app flow

Implemented a new theme, focusing on a modern gray and purple color scheme. Made app usage feel more obvious to the end user, creating a smoother experience. The weight journal entry process was redesigned with a new image option, a change in the way dates are entered (going from from manual to calendar widget), and added a scrollable date list to access historial weight entries.

Enhancement Two

Data Structures and Algorithms

Implemented weight entry analysis with a newly added dashboard

Created a new dashboard page with an algorithm to analyze weight trends and display them in a visually appealing way. The chart uses a Min/Max algorithm to determine the X and Y axis ranges. The chart is created through drawing functionality in Java, drawing a point at each weight entry, and drawing lines between each point. Also created a goal widget, analyzing the user's current weight versus their end goal and color coordinating results based on current progress.

Enhancement Three

Databases

Database restructuring and security improvements

Restructured the database schema to improve performance and maintainability. Implemented security best practices including input sanitization and secure database connection handling. Passwords now handled with encryption, images are stored as URIs in the database to improve performance, and app formats input to ensure database entries are consistent.
